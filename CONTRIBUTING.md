# Contributing

## Guidelines

The site aims to capture the most relevant Java for Education resources for developers, newbies, coding clubs, etc.
General resources are not included, only those that are specific to Java developers.
We aim to include resources that are relevent to learn Java and experiment with tools to understand how to write and execute Java code.
The people who are included should be helping those developers learn how to learn and use Java with public projects, presentations, code examples, libraries, tools, etc.

## Process

1. Make changes to `SPEC.md`
2. Have your AI code assistant update the website from the `SPEC.md` file
3. Validate the changes locally
4. Send a PR
